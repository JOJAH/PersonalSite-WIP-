Projects

Network Topology

This project was the end result of my dissertation project. It is a visualisation of some network traffic data. You can apply various filters to display different information, and clicking on a node or connection will show you further information regarding that element. This project also included a pcap file reader and anayliser, which passes the information into this visualisation, that pcap reader/analyser can be found within the github repo.

Randomised Circles

This was a project to further experiment with D3, I wanted to test regularly updating elements and user interactions with them. Thats why users can click on circles to stop their movement and can open up a menu on the bottom to adjust the number of circles and their speed.

Listed Buildings
